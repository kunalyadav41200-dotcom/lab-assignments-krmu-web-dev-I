# Lab Title: Personal Portfolio Website

## Description
This lab involves creating a responsive personal portfolio website using HTML and CSS.  
The website showcases the student's profile, skills, projects, and contact form.  
It also demonstrates knowledge of modern web design principles such as layout structuring,  
responsive design, box styling, shadows, and clean UI organization.

The project includes the following sections:
- Header with navigation menu  
- Hero section with introduction and profile image  
- About Me section  
- Skills section with skill boxes  
- Projects section with project cards  
- Contact form  
- Footer  

This lab helps students practice front-end development and understand how  
to structure and style a complete website from scratch.
